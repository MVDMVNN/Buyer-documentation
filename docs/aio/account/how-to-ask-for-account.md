# Как запросить аккаунт баеру в AIO

Чтобы запросить новый аккаунт для баера, необходимо поставить задачу с таким содержанием

```html
  Прошу завести нового баера, дать доступы и добавить в команду *НАЗВАНИЕ_КОМАНДЫ*

  Ник баера: *НИК_БАЕРА*
```

В ответ на такую задачу отдел разработки будет отдавать ссылку для регистрации, которую нужно будет передать баеру

Для того, чтобы воспользоваться этой ссылкой и зарегистрироваться по ссылке, необходимо выполнить шаги, которые описаны в этой [статье](how-to-register.md)
