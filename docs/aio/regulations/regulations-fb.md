# Регламент задачи на сетапы Facebook

Для корректного выполнения задачи на создание сетапов в `AIO` под Facebook, необходимо описать её в таком виде

```
Сделать сетап под Facebook
Трекер: AiO
Источник: facebook
Гео: KZ
Оффер: KZ ru OFFER_NAME
Вайты:
WP KZ ru WHITE_1 - 2шт.
WP KZ ru WHITE_2 - 2шт.

Количество: 4шт.

```

* `Трекер` - трекер под который заказываются сетапы
* `Источник` - под какой источник будет делаться сетап
* `Гео` - гео под которое будет делаться сетап, а так же, указываться в кампании, для удобной фильтрации
* `Оффер` - полное название оффера в трекере `AIO`. Название необходимо копировать из самого `AIO` во вкладке `Контент` -> `Лендинги`
* `Вайты` - список вайтов, которые необходимо использовать для сетапов. Если необходимо выставить первый на два домена и второй на других два домена, то возле названия вайта необходимо указать цифру
* `Количество` - это количество запрашиваемых сетапов

!!! info "Совет"
    Если заказывается 2 вайта, но первый вайт нужен на 3 доменах, а второй лишь на 4-м, то можно указать так
    ```
    Вайты:
    WP KZ ru WHITE_1 - 3шт.
    WP KZ ru WHITE_2 - 1шт.
    ```
